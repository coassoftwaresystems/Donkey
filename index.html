Donkey for Oracle

Get yourself a free Oracle database development tool!

Donkey is a free Oracle database development and management tool. It has proven it's value already for years, and is primarily aimed at database developers. It gives you the essential tools to write reliable Oracle database solutions. Using Donkey will help you to write SQL queries, PL/SQL code and explore your database.

You can download Donkey from this page. You choose to download either the stable version, or a test (pre-release) version.

  • Stable version
  • Beta / Test version (use with care, as this version can be unstable)

Donkey is provided, free of charge, by COAS software systems B.V.

When running on Windows 10, you can also get Donkey from the Windows store
